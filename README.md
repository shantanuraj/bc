bc
==

Base Converter program, to convert numbers across diffrent bases.

Usage:

	bc <number> <operation>

Options:

	b - binary
	o - octal 
	d - decmal
	h - hexa-decimal

Example:

	#Input
	bc 2 db 
	#Converts number '2' of decimal base to its binary equivalent.

	#Output
	Input: 2 Output: 10	

Troubleshooting:

	1. Having issues in running?
	1. Most probably a permissions issue, run

		chmod +x bc

	   to fix.
